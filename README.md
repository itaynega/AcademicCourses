# Academic Courses

This repository contains materials, assignments, and projects for various academic courses.

## Courses

### 1. Intro to Machine Learning (Python)
Introduction to machine learning concepts, algorithms, and applications.

**Path:** `intro-to-machine-learning/`

**Setup:**
```bash
cd intro-to-machine-learning
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Digital Image Processing (Python)
Techniques and algorithms for digital image processing and computer vision.

**Path:** `digital-image-processing/`

**Setup:**
```bash
cd digital-image-processing
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Optimization Foundations (MATLAB)
Foundational concepts in optimization theory and numerical methods.

**Path:** `optimization-foundations/`

**Setup:**
- Requires MATLAB R2020a or later
- Optimization Toolbox required

## Directory Structure

Each course folder contains:
- `assignments/` - Course assignments
- `lectures/` - Lecture materials
- `projects/` - Course projects
- `notes/` - Study notes
- Additional course-specific folders

## Version Control

Each course has its own `.gitignore` file configured for the appropriate environment (Python or MATLAB).
