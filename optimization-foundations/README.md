# Optimization Foundations

Course materials and assignments for Optimization Foundations (MATLAB).

## Setup

### MATLAB Requirements
- MATLAB R2020a or later
- Optimization Toolbox
- Global Optimization Toolbox (recommended)

### Getting Started
1. Open MATLAB
2. Set the current folder to this directory
3. Add subdirectories to MATLAB path:
```matlab
addpath(genpath(pwd))
```

## Directory Structure

- `assignments/` - Course assignments and homework
- `lectures/` - Lecture notes and slides
- `projects/` - Course projects
- `notes/` - Personal study notes

## Topics Covered

- Linear Programming
- Nonlinear Programming
- Convex Optimization
- Gradient Descent Methods
- Newton's Method
- Constrained Optimization
- Interior Point Methods
- Genetic Algorithms

## Useful MATLAB Commands

```matlab
% Linear Programming
linprog

% Nonlinear Programming
fmincon, fminunc

% Global Optimization
ga, particleswarm
```
